# Scrapy Commands


### init project
    scrapy startproject stack

### crawl
    scrapy crawl spidername

this __spidername__ is mentioned at spider.py under spiders dir

### crawl and store results
    scrapy crawl stack -o items.json -t json
    -o: output
    -t: type
